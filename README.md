##Full Stack Portfolio Website
This is my personal **Full Stack Portfolio Website** built using React for the frontend and Flask for the backend.  
The website showcases my projects, skills, and contact information.

## Tech Stack

Frontend:
- React
- HTML
- CSS
- JavaScript

Backend:
- Flask
- Python
- REST API

Tools:
- Git
- GitHub

## Installation and Setup

### 1️ Clone the repository
```bash
git clone https://github.com/muskumanasa04/Protfolio.git
```
### 2️ Setup Backend (Flask)

```bash
cd backend
pip install -r requirements.txt
python app.py
```
The backend server will run on:
```
http://localhost:5000
```
### 3️ Setup Frontend (React)
```bash
cd frontend
npm install
npm start
```
The frontend will run on:
```
http://localhost:3000
```
